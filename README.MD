# 新浪微博android客户端,材料设计风格。
用来练习开发技术
## 实现功能
* 登录
* 查看好友最新动态
* 查看微博详情和评论
* 查看我收到的评论
* 发表微博
* 发表评论
* 搜索用户
* 由于新浪api的各种限制，有些功能没有实现，比如关注用户，查看用户资料等等；
